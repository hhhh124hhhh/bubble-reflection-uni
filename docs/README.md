## 🚀 快速开始

1. **克隆项目**
   ```bash
   git clone https://github.com/your-username/bubble-reflection.git
   cd bubble-reflection
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **配置环境**
   - 创建腾讯云开发环境
   - 修改 `src/utils/cloudbase.ts` 中的环境 ID
   - 开启必要的登录认证方式

4. **启动开发**
   ```bash
   # H5 开发
   npm run dev:h5
   
   # 微信小程序开发
   npm run dev:mp-weixin
   ```
